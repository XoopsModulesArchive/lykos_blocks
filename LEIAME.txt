Módule Lykos Blocks

Contém os seguintes blocos:
- últimos x artigos em uma dada categoria
- melhores x artigos em uma dada categoria
- um bloco com estatísticas do site

Suporte para WFSection 1.01 adicionado
Bloco "Parceiro Aleatório" foi removido por esta função já estar disponível no código base do XOOPS.
Renomeou-se 'artigos' para 'notícias', e passou-se a chamar 'wfsections' de artigos.
Há a questão de os blocos ficarem estranhos quando há menos de dois ítens em cada módulo devido ao uso de plurais ('notícias', etc), mas o módulo foi feito na esperança de que você vai postar alguma coisa no seu site antes torná-lo público ;-)

----==== Requerimento ====----

Xoops 2.* ou mais recente

----==== Instalação ====----

1. Envie a paste contida no zip para a pasta de módulos do Xoops.
2. Entre no seu site como Administrador, vá até a administração de módulos e instale o Lykos Blocks. Depois você já pode usar os blocos que achar necessário.
3. O conteúdo de cada bloco é bastante personalizável via templates. Para mudar a forma que as informações são exibidas, altere os arquivos html na pasta templates/blocks/.
